# BGM ファイル

このディレクトリに以下のファイル名で著作権フリー BGM (CC0 / 自作 / ライセンス済み) を配置してください。

| ファイル名 | 雰囲気 |
|----------|-------|
| `calm.mp3` | 落ち着き・解説・教育系 |
| `upbeat.mp3` | 盛り上げ・エンタメ・テンポ重視 |
| `focused.mp3` | 集中・ビジネス・ニュース系 |

**入手元の例:**
- [Pixabay Music](https://pixabay.com/music/) (商用OK)
- [Free Music Archive](https://freemusicarchive.org/)
- [Uppbeat](https://uppbeat.io/)
- [Freesound](https://freesound.org/) (CC0 フィルタ)

ファイルが存在しない雰囲気は自動的にスキップされます。
全部空のままでもアプリは動作します（BGMなしで処理）。

**著作権:** このディレクトリのファイルは git に含まれません（.gitignore 推奨）。
個人利用のみとし、商用利用の場合は各音源のライセンスを確認してください。
