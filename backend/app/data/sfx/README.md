# 効果音ファイル

カット境界に挿入する効果音。CC0 / 自作 / ライセンス済みのファイルを以下の名前で配置してください。

| ファイル名 | 用途 |
|----------|-----|
| `cut.mp3` | カット境界（無音削除・ジャンプカット）に挿入 |

**入手元の例:**
- [Freesound](https://freesound.org/) (CC0 フィルタ)
- [Pixabay Sound Effects](https://pixabay.com/sound-effects/)
- [Mixkit](https://mixkit.co/free-sound-effects/)

**推奨:**
- 短い（0.3〜0.8秒）
- ポップ / スウッシュ / クリック / ディング系
- 音量はファイル側で控えめに（さらにコード側で15%に絞る）

ファイルが無ければ効果音挿入はスキップされます。

**著作権:** このディレクトリのファイルは git に含まれません（.gitignore 推奨）。
個人利用のみとし、商用利用の場合は各音源のライセンスを確認してください。
