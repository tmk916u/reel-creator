# Color Grade LUTs

カラーグレード機能（`color_grade` パラメータ）が使う 3D LUT（`.cube`）置き場。

| ファイル | テイスト | 内容 |
|---|---|---|
| `minimal.cube` | ミニマル | 軽いコントラスト + わずかな彩度低下（雑誌風の落ち着き） |
| `cinematic.cube` | シネマティック | ティール&オレンジ（シャドウ青緑/ハイライト橙） |
| `monochrome.cube` | モノトーン | 彩度0 + わずかな暖色ティント |
| `pop.cube` | ポップ | 彩度・明度ブースト（SNS映え） |

## 出所・ライセンス

全て `backend/scripts/gen_luts.py` による**手続き生成物（CC0相当、著作物非依存）**。
市販LUTのような作り込みはないが、ライセンス上安全。再生成:

```bash
cd backend && python scripts/gen_luts.py
```

## 差し替え

見栄えを上げたい場合は、同じファイル名で外部の `.cube`（`LUT_3D_SIZE` は 17/33/65 いずれも可）を
上書きすればよい。外部LUTを使う場合は出所と再配布ライセンスを必ず確認すること。

FFmpeg `lut3d=file=<path>` で最終焼き込みパス（Stage 7）の映像フィルタ先頭に適用される
（字幕・テロップより前に適用されるため、テキストの色はグレーディングの影響を受けない）。
