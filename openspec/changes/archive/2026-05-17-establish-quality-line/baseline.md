## 不合格項目 → 次に起票すべき change 候補

改訂ベースライン (seitai_food.mov / 994f47eb) で不合格となった項目について、 次の change として起票する候補を優先度順にリスト化:

### 1. ✅ **解決済** 項目#5 + #6: 字幕の語順崩壊と時刻ねじれ
- **change**: `fix-subtitle-word-order-collapse` (commit 426806d, f5ff50e)
- **結果**:
  - 語順崩壊は構造的に解消 (cut2.mp4 を 3段目 transcribe で取り直し)
  - #6 同期 bad_count 18 → 0
  - #5 subword 崩壊なし

### 2. **(致命的)** 項目#2: 冒頭 36 秒の字幕欠落
- **状況**: 動画 84.58 秒中、 字幕が 36.38 秒目から開始。 前半 43% に字幕が無い
- **真因仮説**:
  - 3段目 ReazonSpeech が cut2.mp4 の冒頭発話を認識ミス
  - cut_and_concat の afade(クロスフェード)が冒頭の音量を弱め ASR 精度を低下させている可能性
  - もしくは ReazonSpeech が短い context (84秒) の冒頭で reliability が下がる仕様
- **想定 change 名**: `fix-third-stage-asr-leading-gap`
- **アプローチ案**:
  - (A) 3段目 transcribe で「冒頭 N 秒に word が無い」場合、 2段目 words_cut を該当範囲だけ補完する hybrid 戦略
  - (B) cut_and_concat の冒頭 afade を緩めて ASR 用に音量を保つ(出力動画ではフェード維持)
  - (C) cut2_audio.wav を生成する際にダミーの 0.5秒無音を先頭に挿入して ReazonSpeech のウォームアップを確保
  - (D) ReazonSpeech の initial_prompt に video_context を渡して認識精度を上げる

### 3. **(重要)** 項目#7 + #8: 字幕の切れ目と長さ
- **状況**:
  - 助詞直後切れ 40.9% (旧 13.3% → 悪化、 上限 10%)
  - 8-14 文字: 68.2% (旧 33% → 改善、 目標 70%)
- **想定 change 名**: `fix-subtitle-segmentation-quality`
- **アプローチ案**:
  - words_to_segments の改善: 助詞直後 flush 抑制を 3段目 subword に対しても効くよう拡張
  - max_chars と min_chars の双方を考慮した最適化(短すぎる Dialogue の隣接統合)
  - 句点で flush する条件をより慎重に

### 4. **(任意)** 項目#10 / #12 / #1 などの目視判定
- 残り 2 本のテスト動画(seitai_standard.mov, seitai_long.mov)が配置されてから本格評価
- 1 本だけでは判定者の主観バイアスが大きい

## 改訂ベースラインの所感 (Claude より)

`fix-subtitle-word-order-collapse` の効果は明確に出た:
- subword 語順崩壊が **構造的に消滅** (Stage 5 を 5a/5b に分割し、 cut2.mp4 を 3段目 transcribe する設計に変更)
- #6 タイミング同期は 18件 → 0件で **完全合格**
- 処理時間オーバーヘッドは +7秒のみ(想定 +30-60秒より小さい、 ReazonSpeech モデルキャッシュが効いている)

ただし、 trade-off として **3段目 ReazonSpeech が cut2.mp4 冒頭発話を聞き逃す問題** が顕在化した。 これは元の 1段目 transcribe では起きなかった現象であり、 短い context での認識ミス(または cut_and_concat の afade 影響)が原因と推定される。

次の優先課題は **冒頭欠落の修復** (`fix-third-stage-asr-leading-gap`)。 これが解決すれば 業務量産投入可能ラインに近づく。

業務量産投入可能状態の判断:
- 7/14 (改訂ベースライン) → 目標 10/14 (#2, #7 解決) → 業務投入可
- 14/14 は理想だが、 段階的にラインを上げる方針
