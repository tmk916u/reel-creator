# Reel Creator ロードマップ / 今後の方向性

> ユーザーが示した将来ビジョンの記録。実装は未着手（北極星として参照する）。

## 🌟 North Star（最終理想）

**「動画を入れるだけで、Instagram リールにそのまま出せるレベルの完成品が出てくる」**

さらにその先：

**「アカウントの性質とターゲットを理解した上で、コンテンツを生成できる」**
— 単なる自動編集ではなく、そのアカウントのニッチ／世界観／ターゲット視聴者を踏まえて、
編集トーン・字幕・フック・ハッシュタグ・構成までを最適化する。

---

## 直近のメモ（覚えておくこと）

### 1. テイスト（カラーグレード等）を選べるUXの改修
- 現状: `color_grade` は UI のボタングリッドで選択可（none/minimal/cinematic/monochrome/pop）。
- やりたい改修: より選びやすく。候補
  - 処理前に各テイストの**プレビュー**を見せる（サムネに色味を当てて比較）
  - `subtitle_motion` も同様にプレビュー
  - テイストの追加を簡単に（暖色フィルム / VHS / Y2K など）

### 2. ワンドロップで「IGリール出せるレベル」
- 動画を入れる → 最適なデフォルトで全工程（無音削除・ジャンプカット・字幕・色味・
  リフレーム・フック/CTA）が自動で噛み合い、**そのまま投稿できる完成度**で出力。
- 必要そうなもの:
  - 入力動画の性質を自動判定（トーク主体 / デモ・B-roll主体 / 縦・横）して
    パイプライン設定を自動切替（例: 発話が少ない動画は無音削除を弱め、色味重視に）
  - 低発話動画での字幕ハルシネーション（prompt echo）対策
  - 仕上がりの自動QC（短すぎ・字幕崩れ・無音落ち を検出して警告/再調整）

### 3. アカウント特性・ターゲット理解でのコンテンツ生成
- アカウントの**ニッチ/世界観/ターゲット視聴者**をプロフィールとして持つ
- それに合わせて: キャプション・フック・トピックテロップ・ハッシュタグ・編集テンポ・
  テイスト（色味/字幕モーション）を自動最適化
- さらに先: 既存投稿の傾向学習、トレンド反映、台本/構成提案

## 既に入っている土台（2026-06 時点）
- 無音削除 / 6施策ジャンプカット / WhisperX字幕（キネティック: none/karaoke/fade/pop）
- カラーグレード（テイスト別LUT, 強化版）/ 被写体追従オートリフレーム（YOLOv8, 横→縦）
- バズモード（フック/CTA/トピック/BGM/SFX, LLM連動）/ SNS投稿（IG Reels/Stories, YouTube）
- プリセット（教育系/エンタメ/ぎっしり/ニュース）に色味・字幕モーションも連動

→ 「自動判定＋アカウント文脈」を被せれば North Star に近づく構造は既にある。
